# 对称性与经验公式嵌入报告

## 1. 目标

本报告面向热传导、二维 Burgers、二维不可压 Navier-Stokes 和二维线弹性 PINN。目标是在常规 PDE 残差和数据拟合之外，加入两类领域先验：对称性约束，以及经验公式或简化公式约束。

## 2. 数据生成背景

PDEBench 提供了多类 PDE 数据集和生成代码，包括 2D diffusion-reaction、2D incompressible Navier-Stokes、2D Darcy flow、2D shallow-water 等。Burgers 在 PDEBench 中主要为 1D，二维 Burgers 更常见的做法是用有限差分/Runge-Kutta 数值解或制造解方法生成参考数据。

本仓库第一版采用制造解方法。优点是所有方程都能获得解析真值、源项、边界值和残差校验；缺点是制造解并不一定代表真实工程分布。后续如果需要贴近 PDEBench，可替换 `equations.py` 中的数据生成实现，保留训练接口不变。

## 3. 对称性嵌入方法

设变换为 `T(x)`，输出变量为 `q(x)`。若变量在该变换下满足 `q(x)=s*q(T(x))`，其中 `s=1` 表示偶对称，`s=-1` 表示奇对称，则可加入：

```text
L_sym = mean(|q(x) - s*q(T(x))|^2)
```

代码中对应 `SymmetryResidual`。它适合在 PINN 训练中作为额外 loss 调用。

常见情况：

- 热传导：若几何、源项和边界关于 `x=0.5` 对称，则温度 `u(x,y)=u(1-x,y)`。
- Burgers：若初始速度和边界条件具有镜像关系，则速度分量可能具有不同奇偶性。
- Navier-Stokes：圆柱绕流、通道流等问题经常存在几何对称或反对称。不可压流还可用流函数结构自动嵌入 `u_x+v_y=0`。
- 线弹性：对称载荷与对称几何下，位移、应力、应变分量具有不同奇偶性。

除了 loss 形式，也可以通过网络结构硬编码对称。例如 `u(x,y)=N((x-0.5)^2,y)` 会强制 `u(x,y)=u(1-x,y)`。硬结构更稳定，但只适用于确定无误的对称性；如果真实数据存在扰动，软残差更灵活。

## 4. 经验公式嵌入方法

经验公式可写成 `R_emp(q,x,params)=0`，并加入 `L_emp=mean(|R_emp|^2)`。代码中对应 `EmpiricalFormulaResidual`。

### 热传导

可用公式包括 Fourier 定律 `q=-k grad(T)`、总热阻公式 `Q=Delta T/R_th`、对流换热边界 `-k grad(T).n=h(T-T_inf)`。如果网络只输出温度，可在后处理计算热流；如果网络同时输出温度和热流，则可加入 `q+k grad(T)=0` 残差。

### Burgers

二维粘性 Burgers 是非线性对流-扩散模型。可用先验包括小 Reynolds 数下扩散主导的平滑性、周期边界下能量趋势、对称初始条件下的镜像残差。

### Navier-Stokes

可用公式包括不可压连续性、Stokes 低 Reynolds 近似、Poiseuille 通道流抛物线剖面、Bernoulli 近似，以及阻力系数、Reynolds 数、Nusselt 数等经验关联式。点态微分关系适合直接加入 loss；全局经验关系应先从预测场积分得到宏观量，再加入标量残差。

### 线弹性

可用公式包括几何方程、Hooke 定律、静力平衡、平面应力/平面应变简化、Euler-Bernoulli 梁公式等。本仓库线弹性残差已经包含几何方程、本构关系和平衡方程，因此经验公式模块主要用于加入更高层次工程近似，例如梁挠度、应力集中系数或对称条件。

## 5. 前处理与后处理建议

前处理阶段应根据几何和载荷生成对称点对，对输入坐标做无量纲化，并按经验公式适用条件筛选子域。训练阶段可将 `L_pde + L_data + L_bc + L_sym + L_emp` 作为总损失，并对经验公式使用可调权重。后处理阶段应检查全局守恒量、通量、能量、力平衡，以及经验公式对应的宏观指标。

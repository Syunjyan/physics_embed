# 对称性嵌入方式调研

## 结论

当前代码中的 `SymmetryResidual` 属于“成对点软约束”：对每个点 `x` 构造对称点 `T(x)`，再惩罚 `q(x)-s*q(T(x))`。这是合理的第一版，因为实现简单、适用于镜像/旋转/周期等离散对称，也不会强行假设真实数据完全对称。

但它不是最强的嵌入方式。更推荐按“确定性从强到弱”选择：

1. 对称性确定无误：使用硬嵌入，例如对称坐标映射、周期特征、流函数/势函数结构。
2. PDE 具有连续 Lie 点对称：使用无穷小生成元残差。
3. 对称性可能受边界、载荷、噪声破坏：使用当前的成对点软残差。
4. 数据是网格图像或复杂几何场：考虑群等变网络或 steerable/equivariant layers。

## 方法比较

### 1. 成对点软残差

形式：

```text
L_sym = mean(|q(x) - s*q(T(x))|^2)
```

优点是实现最简单，适合反射、旋转、周期边界、变量奇偶性等先验。缺点是只在采样点上惩罚，不保证网络处处对称；同时需要手动设置 loss 权重。

代码对应：`SymmetryResidual`。

### 2. 硬对称坐标映射

如果已知标量场关于 `x=c` 偶对称，可把输入从 `x` 改为 `(x-c)^2`：

```text
u(x,y) = N((x-c)^2, y)
```

这样网络结构天然满足 `u(c+r,y)=u(c-r,y)`。周期条件可用：

```text
[sin(2*pi*x/L), cos(2*pi*x/L)]
```

这类方法边界/对称严格满足，通常比软约束更稳定，但只适用于确定无误的对称性。

代码对应：`MirrorFeatureMap`、`PeriodicFeatureMap`。

### 3. Lie 点对称残差

许多 PDE 有连续对称群，如平移、缩放、Galilean 变换。文献中的 SPINN / Lie Point Symmetry PINN 将无穷小生成元引出的 invariant surface condition 加入损失函数。形式上，如果生成元为：

```text
X = xi_x d/dx + xi_y d/dy + tau d/dt + eta d/du
```

则可加入：

```text
xi_x*u_x + xi_y*u_y + tau*u_t - eta = 0
```

这比简单镜像残差更适合连续对称，能提高样本效率。但前提是需要解析推导该 PDE 与边界/初值共同保留的对称性。

代码对应：`InfinitesimalSymmetryResidual`。

### 4. 等变网络

对于旋转、平移、反射等群，可用 group-equivariant network、steerable CNN 或 equivariant graph network，使中间特征按群作用变换。这适合规则网格场或几何数据，但比 MLP PINN 更复杂，且需要让自动微分残差与架构兼容。

### 5. 物理结构硬嵌入

这类方法不一定叫“对称”，但本质上是把物理不变量写进结构：

- Navier-Stokes 用流函数 `psi` 表示速度，自动满足 `u_x+v_y=0`；
- 势流可用速度势；
- 线弹性可由位移导出应变，再由 Hooke 定律导出应力，减少自由输出变量；
- 热传导可输出温度势，再通过 Fourier 定律导出热流。

这通常比额外 loss 更可靠，因为它减少了网络需要学习的约束。

## 对四类方程的建议

- 热传导：若源项和边界对称，优先使用 `MirrorFeatureMap` 或直接构造对称 ansatz；周期边界用 `PeriodicFeatureMap`。
- Burgers：周期/平移对称适合周期特征；Galilean 或缩放对称可考虑 Lie 残差，但必须检查初始和边界条件是否破坏该对称。
- Navier-Stokes：继续使用流函数硬嵌入不可压条件；通道流或圆柱绕流的几何对称可叠加软残差或对称采样。
- 线弹性：载荷和几何对称明确时，可对位移分量分别设置奇偶约束；小变形几何方程和 Hooke 定律更适合结构性硬嵌入或强残差。

## 实用策略

建议默认采用组合方案：

```text
硬结构嵌入 + PDE 残差 + 边界损失 + 低权重软对称残差
```

只有当对称性来自严格几何和边界条件时，才把它做成硬嵌入；如果只是经验上近似对称，应保留为低权重软约束或后处理指标。
